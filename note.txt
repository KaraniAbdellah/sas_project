#define & #endif
	Les directives #ifndef, #define, et #endif créent une "protection"
	pour que le contenu du fichier ne soit inclus qu'une seule fois.


# Makefile
	Un Makefile en C est un fichier texte qui contient des instructions pour
	compiler un programme. Il automatise la création des fichiers
	exécutables à partir du code source.


# Constance & Define
	Une constante (const) est liee a un type, donc verifiee par le compilateur.
	Avec #define, on crée une macro sans type qui remplace chaque occurrence
		du nom par sa valeur au moment de la compilation
	

# printf(GREEN "\n✅ Tache bien tries\n" RESET);
	Ce code affiche le texte "✅ Tache bien triée" en vert grâce à GREEN.
	RESET remet la couleur à la normale après l'affichage.


# CFALGS
	Utiliser CFLAGS rend le code plus propre et personnalisable lors de la compilation.
